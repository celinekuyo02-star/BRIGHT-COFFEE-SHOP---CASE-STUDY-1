# BRIGHT-COFFEE-SHOP---CASE-STUDY-1
Conducting sales analyses for the CEO to understand the performance of the shop in the last six months.

### OVERVIEW
This repository contains a comprehensive case study conducted as part of Brightlearn 2026 program.
It focuses on deriving actionable insights and data-driven recommendations.

### Methodology $ process
Project planning, Data Acquisistion, Data processing, Analysis and Visualization is the strusture I followed.

### Tools used 
Planning:Miro and Project Gantt chart
Data processing: SQL
Analysis: Miscrosoft excel
Presentation:Powerpoint

### Insights
Products with most revenue.
what time of day the store performs best.
Sales trends across products and time intervals.
Recommendations for improving sales performance.
